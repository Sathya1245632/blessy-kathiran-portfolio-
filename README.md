# Blessy Kathrin A - Portfolio 🚀

A high-end, premium developer portfolio for Blessy Kathrin A, featuring a dark-luxury aesthetic with cinematic animations and a conversion-focused layout.

## ✨ Features

- **Futuristic Dark Aesthetic**: Implemented with a custom dark mode palette and glassmorphism.
- **Cinematic Hero Section**: High-impact introduction using Framer Motion and modern typography.
- **Project Case Studies**: Engaging showcases for accounting and development projects.
- **Performance Optimized**: Built with Next.js App Router and optimized for fast page loads.
- **Fully Responsive**: seamless experience across mobile, tablet, and desktop devices.

## 🛠️ Built With

- **Framework**: [Next.js](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Components**: [Radix UI](https://www.radix-ui.com/) & [Shadcn/UI](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.x or later)
- npm or yarn

### Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Sathya1245632/blessy-kathiran-portfolio-.git
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    # or
    yarn dev
    ```

4.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
